# Breadth First Search (BFS) in Python

This project provides a clean, modular implementation of the Breadth First Search algorithm. 

## Project Structure
- `src/`: Contains the core logic (Graph class and BFS algorithm).
- `main.py`: A demonstration script showing how to use the implementation.
- `tests/`: Basic unit tests for the algorithm.

## How to Run

### Prerequisites
- Python 3.x

### Execution
Run the demonstration with:
```bash
python main.py
```

### Testing
Run tests with:
```bash
python -m unittest discover tests
```
